# Worked Example — Consumer Launch

**Project:** TapIn  
**Archetype:** Consumer Magnet

## Core story angle

Student organizers should not need a wallet tutorial just to collect dues or manage access to an event.

## Example post

Most campus event flows are still a pile of screenshots, bank transfers, and “send me proof you paid.”

TapIn makes that loop cleaner:

- collect dues
- issue a QR pass
- verify entry instantly

The point is not “put clubs onchain.”

The point is that student organizers need less admin and fewer excuses.

If you run events on campus, DM us. We want to test this with real clubs, not just demo judges.
